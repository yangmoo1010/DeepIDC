# DeepIDC
This is a a prediction framework of injectable drug combination (IDC) based on Heterogeneous information and deep learning, as described in our paper [DeepIDC](). <br>

## Requirements
* keras == 2.6.0
* tensorflow == 2.6.0
* numpy == 1.19.2
* pandas == 1.3.3
* scikit-learn == 0.24.2
## Run the demo
```bash
python DeepIDC.py 
```
